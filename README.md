## 笔记内容/记录方法

在每个难度的文件夹，每道题我都会创建一个 jupyter notebook文件，其中记录了 “题目要求” 以及 “代码”。

代码部分我会收录**自己的第一遍解法**以及网上的**优质解法**，并**总结考点**。

之后刷的题多了，也会按考点类别总结题目，写下新的.md文件。
